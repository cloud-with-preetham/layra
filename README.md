# Layra 🎧

Layra is a modern Spotify-inspired music streaming platform built with a microservices architecture and DevOps-first infrastructure.

## Tech Stack

Frontend

- Next.js
- TailwindCSS

Backend

- Node.js
- Express
- PostgreSQL

Infrastructure

- Docker
- Kubernetes
- GitHub Actions

Monitoring

- Prometheus
- Grafana

## Features

- Music streaming
- Playlists
- User authentication
- Search
- Recommendations
